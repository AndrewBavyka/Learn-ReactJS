Базовая теория

React - это библиотека на основе языка программирования JavaScript. Она разработана компанией Facebook которая должна упростить разработку пользовательских интерфейсов.
Сама концепция React заключается в разработке SPA (Single Page Application) - приложений которые ведут себя крайне отзывчивы. 
Главная задача - это изменение страниц без перезагрузки.
React контролирует использование памяти в компьютере.

Императиный подход говорит о том "Как именно дойти к результату" конкретные шаги
Пример: Приходим с другом  в кафе и вы говорите что: "тот угловой столик свободный, мы сейчас пойдем туда и сядем там"
Описывем конкретные шаги

Плюсы React

+Декларативный подход говорит нам "Какой результат нам нужен"
Пример: Приходим в кафе и говорим просто официанту "Столик на двоих пожалуйста"

+Компоненты элементы которые повторяются но их можно переиспользовать
